# Keyboard Shortcuts

Quick reference for keyboard shortcuts in the Personal Memory System.

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `⌘K` / `Ctrl+K` | Open global search |
| `Escape` | Close modal/dropdown |

## Navigation

| Shortcut | Action |
|----------|--------|
| `Tab` | Move to next field |
| `Shift+Tab` | Move to previous field |
| `Enter` | Submit form / Send message |

## Chat Interface

| Shortcut | Action |
|----------|--------|
| `Enter` | Send message |
| `Shift+Enter` | New line in message |
| `↑` | Edit last message (when input empty) |

## Memory List

| Shortcut | Action |
|----------|--------|
| `↑` / `↓` | Navigate memory list |
| `Enter` | Open selected memory |
| `Delete` | Delete selected memory |

## Modals

| Shortcut | Action |
|----------|--------|
| `Escape` | Close modal |
| `Enter` | Submit (in forms) |

## Text Editing

| Shortcut | Action |
|----------|--------|
| `⌘A` / `Ctrl+A` | Select all |
| `⌘C` / `Ctrl+C` | Copy |
| `⌘V` / `Ctrl+V` | Paste |
| `⌘Z` / `Ctrl+Z` | Undo |
| `⌘⇧Z` / `Ctrl+Shift+Z` | Redo |

## Search

| Shortcut | Action |
|----------|--------|
| `⌘K` / `Ctrl+K` | Focus search bar |
| `Enter` | Execute search |
| `Escape` | Clear search / Close |

## Tips

- On Mac, use `⌘` (Command) key
- On Windows/Linux, use `Ctrl` key
- Most browser shortcuts also work (e.g., `⌘F` for find)

## Accessibility

The interface supports:
- Full keyboard navigation
- Screen reader compatibility
- Focus indicators
- Skip links
