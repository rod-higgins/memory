## About

The Personal Memory System (PMS) enables you to create AI that truly understands you. By storing your preferences, knowledge, and context, every AI interaction becomes more relevant and personalized.

**Key Features:**
- Three-tier memory hierarchy
- Multiple AI provider support
- Automatic data ingestion
- Complete privacy (all data stays local)
- Multi-user support with isolation
